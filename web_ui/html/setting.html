快捷菜单

    请按遥控器"菜单"键打开快捷菜单。在快捷菜单开启的状态下，再次按"菜单"键可将其隐藏。其功能主要包括：后退、刷新、主页、收藏、收藏夹、打开、退出。其中"打开"为手动输入网址的位置

输入法

1）输入法的开启和关闭
    当用户将光标移动至页面中需要输入字符的地方并按下遥控器"OK"键，输入法将自动开启，否则输入法将自动关闭。

2）输入法的相关按键
    打开输入法即为虚拟键盘，默认为数字键盘，通过遥控器方向键选中您想输入的内容,按"ok"键确定输入，如输入错误则选择"<-"键来删除，或者选择"Reset"键重新输入，输入完成则选择"Enter"键完成输入，此时虚拟键盘将自动关闭

3）输入法的不同状态
        如果选中虚拟键盘中的"123"键，则键盘处于"数字"状态，直接选择选中想输入的内容按"ok/确定"键;
    如果选中虚拟键盘中的"abc"键，则键盘处于英文状态，直接选择选中想输入的内容按"ok/确定"键;
    如果选中虚拟键盘中的"中文"键，则键盘处于汉字状态，汉字输入为汉语拼音拼写，选择字符拼出您想输入的字，选择后按"ok/确定"键;
    如果选中虚拟键盘中的"#+="键，则键盘处于特殊符号状态，直接选择选中想输入的内容按"ok/确定"键
